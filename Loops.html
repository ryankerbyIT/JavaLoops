Introduction

Loops are essential structures in Java that allow you to execute a block of code repeatedly. Depending on the condition provided, the loop will continue executing as long as the condition is met. Java offers several types of loops, such as while, do-while, for, and the enhanced for loop. Each loop is used for specific situations and optimizations. Let's explore them in detail.

While Loop

The while loop is one of the most basic loops in Java. It checks the condition before the block of code is executed. If the condition evaluates to true, the loop runs; otherwise, it terminates.

Syntax:


while (condition) {
    // code block to be executed
}

Example:


int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

Output:


0
1
2
3
4

The above code prints numbers from 0 to 4, incrementing i each time. The loop runs as long as i is less than 5.

Do-While Loop

The do-while loop is similar to the while loop but differs in one key aspect: it guarantees that the block of code is executed at least once, even if the condition is false.

Syntax:


do {
    // code block to be executed
} while (condition);

Example:


int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);

Output:


0
1
2
3
4

This loop behaves like the while loop, except it prints at least once, even if the condition is not met initially.

For Loop

The for loop is used when you know the number of iterations in advance. It allows you to define the initialization, condition, and increment/decrement in a single line, making it very powerful for iterating over a range of values.

Syntax:


for (initialization; condition; increment/decrement) {
    // code block to be executed
}

Example:


for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Output:


0
1
2
3
4

The for loop runs exactly 5 times, starting at i = 0 and increasing i by 1 each iteration until i < 5.

Enhanced For Loop

Also known as the for-each loop, the enhanced for loop is used primarily for iterating through arrays or collections. It eliminates the need for a counter or index variable, making it simpler and safer to use when you want to access each element of an array or collection.

Syntax:


for (type variable : array/collection) {
    // code block to be executed
}

Example:


int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}

Output:


1
2
3
4
5

The enhanced for loop iterates through each element in the array and prints it, simplifying the loop operation for arrays and collections.

Use Cases

Loops are useful in a wide range of programming scenarios, including:

  • Iterating through elements in an array or collection.
  • Executing code multiple times until a condition is met.
  • Building infinite loops for background tasks (with exit conditions).
  • Processing user inputs or reading files until the end of the file is reached.

Choosing the right loop type depends on the situation and how the loop condition is structured.
